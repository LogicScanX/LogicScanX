# Logic ScanX

Multi Advanced Tool v2.0 for Termux.

## Installation

```bash
termux-setup-storage
pkg update -y && pkg upgrade -y
pkg install git python -y
git clone https://github.com/akashdeepx/HostScanX.git
cd HostScanX
bash install.sh
python LogicScanX.py
```

## Features
- Host Scanner
- CIDR Scanner
- Domain Extractor
- Multi CIDR Scanner
- Multi Port Scanner
- Subdomain Hunter
- File Splitter
- CIDR To Domain
- Remove Duplicate Domains
- Reverse IP Lookup
- Host Information
